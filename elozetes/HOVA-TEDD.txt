GITHUBRA FELTÖLTÉS:
-------------------
1. Menj ide: https://github.com/Kovrat12345678/kodolt-ero/tree/main/elozetes
2. "Add file" > "Upload files"
3. Drag & drop a videót
4. "Commit changes"

Vagy parancssorból:
  cd "C:/xampp/htdocs/Zynox film"
  git add elozetes/elozetes.mp4
  git commit -m "Előzetes feltöltése"
  git push

Támogatott formátumok:
- elozetes.mp4  (ajánlott, jól játszik mobilon is)
- elozetes.webm
- elozetes.mov

⚠️  GitHub limit: max 100 MB / fájl
